## GitSearch
 - run npm i to download all the pkg necessary to run the app .
 - run ng serve to lunch the app .
## Repository Page
 - In the first page, you will find the serach box, where you can type : 
  - Repository Name
  - Stars
  - Language of the repository you are looking for (Javascript, Ruby , Python ... )
 - Click on search to start searching for it, and wait for the results. 
 - By clicking on a specific row, you can navigate to that specific Repository information, and by clicking on `Search for Commits` have a view on the commits of it .
 
 
Enjoy .
